Para <strong>obras civis nacionais de longo prazo</strong>, o <strong>BNDES é insubstituível</strong> pelo prazo de 16 anos e carência de 5 anos. A estratégia ótima: <strong>BNDES para a base (50-60%) + BID Invest/Bonds para o topo (40-50%)</strong>.
    </div>

        <strong>Referências Regulatórias:</strong> Circular SUP/ADIG nº 36/2026-BNDES, itens 4.3.1 (Taxas de Juros), 5.4.1 (Taxas Projetos), 9.2 (Limite Compartilhado) | BID Invest Sustainable Debt Framework, seções 2.1 (Use of Proceeds), 3.2 (Sustainability-Linked Bonds) | IFC Performance Standards 1-8

            <div class="callout-title">Por que o SLB é a "Joia da Coroa"</div>

Como o BIM já simula isso com precisão, o risco de o hotel errar a meta e sofrer penalidade é <strong>zero</strong>. O dinheiro entra como caixa livre, mas o custo cai conforme o projeto entrega resultados.
